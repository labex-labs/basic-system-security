Aprenda Segurança Básica do Sistema. Este módulo abrange Configuração do Firewall UFW, Auditoria de Logins de Usuários, Memória Compartilhada Segura no Tmpfs, Compreensão da Configuração do Sudoers, Verificação de Portas de Escuta para Auditoria de Segurança. Você dominará essas habilidades essenciais do Linux através de laboratórios práticos e desafios do mundo real.
