基本的なシステムセキュリティについて学びます。このモジュールでは、UFW ファイアウォールの設定、ユーザーログインの監査、Tmpfs 上のセキュアな共有メモリ、Sudoers 設定の理解、セキュリティ監査のためのリッスンポートの確認について扱います。実践的なラボと実世界の課題を通じて、これらの必須の Linux スキルを習得します。
